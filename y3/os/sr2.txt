1) Що означають оператори в командному рядку:
;	оператор послідовного виконання програм.
&	оператор паралельного виконання програм у фоновому режимі.
&&	друга команда виконується за умови успішного виконання першої.
||	друга команда виконується за умови неуспішного виконання першої.

2) Які номери, назви та яке призначення мають стандартні потоки вводу та виводу?
0	stdin - потік введення.
1	stdout - потік виведення.
2	stderr - потік помилок.

3) Які функції виконують наступні оператори перенаправлення вводу/виводу?
>	перенаправлення потоку (зазвичай виведення) в режимі перезапису.
<	перенаправлення потоку (зазвичай введення) в режимі читання.
>>	перенаправлення потоку (зазвичай виведення) в режимі дописування.
|	перенаправлення потоку до іншої програми.

4) Як перенаправити потік помилок програми prog у стандартний потік виводу при запуску? Написати командний рядок.
prog 2>&1

5) Як створити архів каталогу /home/user1/ за допомогою програм tar та gzip? Написати командний рядок.
tar -cf user1.tar home/user1/
gzip -r home/user1/

6) Як створити tar-gzip архів каталогу /home/user1/ за допомогою програми tar? Написати командний рядок. 
tar -czf user1.tar.gz home/user1/

7) Що роблять наступні скрипти shell? Який їх результат роботи?

if [ -e somefile.txt ] ; then
cat somefile.txt
else
echo "no such file"
fi

Якщо існує файл з назвою "somefile.txt", тоді вивести його вміст, інакше вивести "no such file".

----------------------------------------------------------------------------------------------------

for a in 1 2 3 4 5; do

touch file$a

done

Для кожного a зі списку [1,2,3,4,5] створити файл з назвою "file" + a

----------------------------------------------------------------------------------------------------

for a in $( seq 1 10 ) ; do

cat file_$a

done

Для кожного a зі списку [1...10] вивести зміст файлу з назвою "file_" + a

----------------------------------------------------------------------------------------------------

while [ -d mydirectory ] ; do

ls -l mydirectory >> logfile

echo -- SEPARATOR -- >> logfile

sleep 60

done

Поки існує директорія "mydirectory", вивести інформацію про неї в файл logfile, вивести "-- SEPARATOR --" у файл logfile, чекати 60 мс, повернутися на початок циклу.
